##Local Dev Setup

* Install Node.js (of course)
* Install NW globally: npm install -g nw
* Install the Grunt CLI globally: npm install -g grunt-cli


##Run the project
###USING NW

Navigate to the project directory
`> nw`


###OLD DIRECTIONS USING NODE WEBKIT

Navigate to the project directory
`> nodewebkit app`
